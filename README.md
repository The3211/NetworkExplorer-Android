# Network Explorer Android App

This is a simple Android application built with Kotlin, designed to scan for SMB network devices, browse shared folders, and transfer files.

## Features
- Scan for SMB devices on the local network.
- Browse and view shared folders.
- Transfer files between the device and shared folders.

## Setup
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on an Android device or emulator.
